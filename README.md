# LiveMarket (Paper 1.21.1 + Vault) — v1.3.4

## Фичи
- `/shop` GUI: 6 категорий (armor/food/tools/ores/blocks/misc)
- Склад: количество товара зависит от покупок/продаж (товар заканчивается)
- Если stock = 0 — купить нельзя, пока игрок не продаст
- Продажа учитывает прочность (сломанное продаётся дешевле)
- Если предмет часто продают — sell-выручка дополнительно снижается
- Ежедневный пересчёт цен в 22:00 (Europe/Moscow) + сообщение в чат
- Таймер до обновления (HH:mm:ss) в GUI, обновляется раз в секунду
- Кнопка `/ah` внизу меню
- Цена покупки не растёт (верхний предел = basePrice)
- Сообщения о покупке/продаже показывают НАЗВАНИЯ ПРЕДМЕТОВ НА РУССКОМ

## Зависимости
- Vault
- Любая экономика (EssentialsX, CMI, ...)

## Генератор предметов
- Создаёт файл `plugins/LiveMarket/generated-items.yml`

## Редактор ассортимента
- Команда: `/shopedit` (permission `livemarket.shop.edit`)
- В редакторе: предмет на курсоре в слот — добавить; Shift+ПКМ по предмету — удалить

## Лимит склада
- Продажа запрещена, если stock достиг stockTarget (например 320/320)

## Команды и права
- `/shop` — открыть рынок (perm `livemarket.shop`)
- `/price <material>` — цена предмета (perm `livemarket.price`)
- `/shopedit` — редактор ассортимента (perm `livemarket.shop.edit`)
- Категории: `livemarket.category.<key>` (например `livemarket.category.food`)
