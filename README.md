# LiveMarket (Paper 1.21.1 + Vault)

Живой рынок с 6 категориями, складом (stock), динамическими ценами и ежедневным обновлением в 22:00 по Москве.

## Фичи
- `/shop` GUI: 6 категорий (armor/food/tools/ores/blocks/misc)
- Склад: количество товара зависит от покупок/продаж
- Если stock = 0 — купить нельзя, пока игрок не продаст
- Продажа учитывает прочность (сломанное продаётся дешевле)
- Если предмет часто продают — sell-выручка дополнительно снижается (sell pressure)
- Ежедневный пересчёт цен в 22:00 (Europe/Moscow) + сообщение в чат
- Кнопка `/ah` внизу меню

## Сборка на GitHub
В репозитории есть workflow `.github/workflows/build.yml` — он собирает jar и прикрепляет артефакт.

## Зависимости
- Vault
- Любая экономика (EssentialsX, CMI, ...)

## Permissions
Все permissions перечислены в `plugin.yml`, их описание — в `config.yml` (permissionsDescriptions).

- Микро-обновление цены после каждой сделки (чтобы sell/buy реагировали сразу)
